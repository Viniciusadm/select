MinSelect — showcase interativo

Uma biblioteca leve em TypeScript para selects ricos: busca com acentos, virtualização, portal, optgroup, e uma API ergonômica. Cada seção abaixo demonstra uma capacidade — abra o <details> em cada exemplo para ver o código fonte.

01 Live search com acentos

Busca normaliza diacríticos automaticamente — digite sao e São Paulo aparece. Atalho de teclado: comece a digitar com o select fechado para type-ahead.

HTML
<select class="selectpicker"
        data-live-search="true"
        data-live-search-placeholder="Buscar cidade…">
  <option value="sp">São Paulo</option>
  …
</select>

02 Optgroup & disabled novo

Suporte nativo a <optgroup> com cabeçalhos e a <option disabled> — opções desabilitadas são puladas pela navegação de teclado.

HTML
<optgroup label="Funcionais">
  <option value="hs">Haskell</option>
  <option value="ocaml" disabled>OCaml (em breve)</option>
</optgroup>

03 Subtext & tokens novo

Cada opção pode ter texto secundário (data-subtext) e palavras-chave extras de busca (data-tokens) que casam sem aparecer no rótulo. Tente buscar por sampa, carioca ou capital.

HTML
<option value="sp"
        data-subtext="São Paulo / BR"
        data-tokens="sampa paulistano capital">
  São Paulo
</option>

04 Cascade dinâmico (city → bairro)

Modifique o <select> nativo e chame refresh() — o picker re-sincroniza sem perder o estado de busca/foco.

0 bairros
JS
// modifica o <select> nativo e refaz a UI
hood.innerHTML = "";
list.forEach(h => hood.appendChild(opt(h)));
MinSelect.get(hood).refresh();

05 Multi + actions box + API completa

Multi-select com botões "selecionar/limpar tudo" e "Aplicar" (data-actions-box) e formato em contador (data-selected-text-format="count"). O botão Aplicar dispara applied.sl.select com os valores atuais e fecha o dropdown — útil para confirmar a seleção sem reagir a cada clique. Os botões abaixo exercitam toda a API pública.

JS
const picker = MinSelect.get(el);
picker.val(["js", "go"]);     // set
picker.val();                  // get → string[]
picker.selectAll(); picker.deselectAll();
picker.show(); picker.hide(); picker.toggle();
picker.refresh(); picker.destroy();
